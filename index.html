Neo's CheatSheet

Git

git init

Dieser Befehl erstellt ein leeres Git Repository

git add .

Alle veränderten und neu erstellen Dateien werden zum Commiten vorgemerkt

git commit -m 'First Commit'

Die vorgemerkten Dateien werden commited. Durch -m kann ein der notendige Kommentar gleich im selben Zug erstellt werden.

git remote add origin git@github.com:GithubUserName/GithubRepository.git

Fügt origin als Remote Repository hinzu. Nach origin folgt die url des zu nutzenden Repositories

git push

Alle lokalen commits werden in das remote repository gepusht

Terminal

mkdir pfad/ordnername

Erstellt am angegebenen Pfad einen neuen Ordner

cd pfad/ordnername

Wechselt in den angebenene Ordner

touch pfad/dateiname.endung

Erstellt eine neue Datei im angegebenen Pfad

ls -la

Listet den Inhalt des aktuellen Ordners auf. Durch die Parameter l und a werden alle Elemente in einer ausführlichen Listenansicht dargestellt und auch alle versteckten Elemente werden dargestellt.
